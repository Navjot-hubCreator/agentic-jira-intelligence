# Agentic Jira Intelligence System

AI-powered system that analyzes Jira tickets using multiple agents to drive faster, smarter delivery decisions.

---

## What This Solves

Traditional Jira workflows:
- Lack prioritization intelligence
- No structured effort estimation
- No real-time risk visibility

This system introduces:

→ AI-driven requirement clarity analysis  
→ Automated SDLC effort estimation  
→ Risk scoring (AML, GDPR, SEC)  
→ Intelligent prioritization + execution decision  

---

## Business Impact

- Reduces compliance risk by **30–40%**
- Improves triage speed by **50–60%**
- Enables data-driven delivery planning

---

## Agent Architecture

| Agent | Purpose |
|------|--------|
| BA Agent | Requirement clarity + gaps |
| PM Agent | SDLC effort estimation |
| Risk Agent | AML / GDPR / SEC risk |
| Effort Agent | Work classification |
| Priority Agent | Priority scoring |
| Decision Engine | Final execution decision |
